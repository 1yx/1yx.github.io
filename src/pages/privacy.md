## 隐私政策 (Privacy Policy)

本着“最小化数据采集”的原则，本站致力于为您提供一个干净、无追踪的阅读环境。

### 1. 网站统计与分析
本站 **不使用** 任何侵入式的第三方追踪代码（如 Google Analytics），也 **不使用** 任何追踪性 Cookie 来记录您的互联网行为。因此，您在本站不会看到烦人的 Cookie 同意弹窗。

本站的访问流量统计由 **Cloudflare Web Analytics** 提供。这是一款极其注重隐私的分析工具，它仅收集以下匿名的聚合数据，用于评估网站的运行健康度：
* 页面访问量 (PV) 与独立访客数 (UV)
* 访客的粗略地理位置（国家/地区级别）
* 浏览器类型与设备类型
* 页面加载性能核心指标 (Core Web Vitals)

这些数据无法用于识别您的个人身份，也不会与任何第三方共享用于广告营销。

### 2. 服务器日志
本站托管于 GitHub Pages，并由 Cloudflare 提供 CDN 加速与安全防护。为了保障服务器的稳定运行和防御恶意攻击（如 DDoS），这些基础服务提供商可能会在网络边缘节点自动记录标准的服务器日志（可能包含 IP 地址、访问时间、User-Agent 等）。这些日志仅出于网络安全目的短期保留。

### 3. 第三方链接
本博客的文章中可能会包含指向其他第三方网站的链接（如 GitHub 仓库、官方文档或其他博客）。当您点击这些链接离开本站后，您的隐私将受该第三方网站隐私政策的约束，本站对其隐私做法不承担责任。
