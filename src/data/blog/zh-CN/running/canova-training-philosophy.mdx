第一篇主要包含 Renato Canova 对训练分类、速度区间和具体 workout 的解释；第二篇更多是论坛对 Canova 体系的归纳、争论和补充，尤其是"不要照抄训练日志，而要理解哲学"的角度。

## 核心判断

Canova 体系的核心是以比赛速度为中心，逐步把跑者的能力延展到目标比赛所需的距离和持续时间。

Canova 在第一篇帖子里说，中长跑到马拉松，本质上都是专项延展（specific **extension** ）。他把训练分为四类：

- **Regeneration** ：恢复训练。
- Fundamental：基础有氧训练。
- Special：辅助专项训练的支撑训练。
- Specific：对比赛表现产生直接影响的专项训练。

这样的分类很关键，因为它没有按照"轻松跑、节奏跑、间歇跑"这种传统的类丹尼尔斯训练的方式分类，而是以服务于比赛目的来进行的分类。

Canova 的 专项课（Specific）的目标在于，能在多大程度上支持目标比赛速度，以及是否能支持比赛的距离。而其他的训练课安排的核心逻辑都是以帮助专项课达成此目标为目的的。

## Regeneration：恢复不是垃圾跑

在普通的中长跑教练看来，某个水平的运动员需要满足某个特定数值的月跑量，刨除掉强度课程之外，需要见缝插针式的安排恢复跑，以凑够月跑量。但在 Canova 体系里恢复跑（再生跑 Regeneration）则带有明确的目的：让身体从关键训练中恢复过来。带着恢复的目的，找到身体轻盈、舒适、流畅、通透的感觉。

Canova 提到，恢复跑的速度通常大约是阈值速度的 60-70%。但不要把重点放在配速，要清醒地将恢复跑的目标放在服务于下一次真正训练。

这和大家常说的"垃圾跑量毫无意义"不同。Canova 明确表示恢复跑不能直接提升比赛能力，它的作用在于能让跑者承受更高质量的专项课。“垃圾跑量”的概念只在这一种情况下成立：专项课前，安排恢复跑计划与完全休息的计划做对比，前者的恢复情况反而更差。

第二篇讨论中有人进一步解释：Canova 系统高度依赖 hard/easy modulation，也就是强弱调制。关键专项课之前身体要有足量的储备，之后要有足够恢复，全部训练的 60-80% 的跑量都可以是相对舒服的恢复跑。

## Fundamental：有氧基础不是永远慢跑

Fundamental 是 基础有氧训练（basic aerobic training）。这种课的目标是支撑更高强度训练。

Canova 对 10K、半马、全马给过基础有氧训练的时间和速度范围：

| 项目 | Fundamental 持续时间 | 速度范围 |
|---:|---:|---:|
| 10K | 1h-1h30 | 约比赛配速的 80%-87% |
| 半马 | 1h20-1h40 | 约比赛配速的 80%-87% |
| 全马 | 1h45-2h30 | 约比赛配速的 83%-90% |

Fundamental 课的目标 是最大化有氧适应，有几个原则：
- 尽量避免糖代谢主导
- 尽量避免乳酸堆积
- 尽量避免恢复债
 
但是要注意，针对已经训练多年的成熟跑者，单纯 LSD 带来对跑步能力的提升效果会边际递减，如果想继续提升有氧能力，则需要提高强度。

针对强度提高的幅度，更准确地说，是在基础期后段，逐渐把一部分长距离 从 有氧耐力区 推向 有氧动力区。要满足前面所言的原则，尽可能贴近乳酸阈值配速，但不要超过它。

## Special：专项的支撑训练

Special 是 Specific 的直接支撑。Canova 把 Special 分成两类：

- Biomechanical support（生物力学支撑训练）：力量、短上坡、比比赛速度快很多的短距离。
- Bioenergetic / metabolic support（供能代谢系统支撑训练）：大约目标比赛速度 90% 附近的延展训练。

这里最容易误解的是马拉松。Canova 明确指出：

> 对马拉松来说，Special 往往是比马拉松短、但比马拉松快的训练；对其他项目，Special 往往是总量更长、速度更慢的训练。

之所以这样安排，是因为马拉松训练和 5K/10K/半马 训练 的最大区别：马拉松训练不会练到比 比赛距离 更长的距离。

这也解释了为什么马拉松运动员也会做很快的 1K、2K、3K、5K 重复跑。这些更短距离的训练绝非是为了培养 5 公里路跑的能力，而是为了能让身体更好地支撑住马拉松配速，让马配变得更轻松。

## Specific：真正决定比赛表现的专项课

Canova 对 Specific 的定义很明确：

> 比赛速度 95-105% 区间内的训练。

也就是说，Specific 完全是围绕比赛速度的训练。

- 对 10K，是 10K race pace 附近。
- 对半马，是 半马配 HMP 附近。
- 对全马，是 马配 MP 附近。

Specific 课程安排的进展方式并非简单地跑得更快，而是 扩展（ **extension** ）：在接近目标比赛速度的前提下，逐步延长单段距离、总量，或缩短恢复。

第二篇讨论中也总结得很直接：Canova 的课程进展（progression）重点就是扩展（extension），长间歇的距离在同一速度下变得更长。

Canova 式专项课的进步可以是：

- `10 x 1km` 变成 `5 x 2km`
- `5 x 3km` 变成 `3 x 5km`
- `25km @ 97% MP` 变成 `30km @ MP`
- 同样配速下，恢复从慢 jog 变成 **float** （较快但受控的恢复跑）
- 同样总量下，后半段更接近比赛节奏

这比 “固定距离，但每周都跑更快” 更接近他的体系。

## 周期观：从远到近

Canova 的周期不是传统固定周结构。他更关心训练距离和比赛距离之间的关系。

可以理解成：

| 阶段 | 目的 | 训练特征 |
|---|---|---|
| Introductive | 恢复一般体能 | 长跑、力量、短上坡、技术 |
| Fundamental | 建立高水平有氧支撑 | 最大跑量，逐渐提高速度 |
| Special | 支撑专项 | 对半马及以下距离，发展比赛速度上下两侧的能力；对全马，重点发展快于马配的支撑能力 |
| Specific | 直接转化比赛能力 | 95-105% race pace，延展距离 |
| Taper | 保留专项感觉，释放疲劳 | 降量，不完全丢掉速度 |

Canova 提到，6 个月大周期（macrocyle）可以分成不同中周期（mesocycle）。Fundamental 期通常可持续约 2 个月，并且这期间会达到最高跑量，同时强度逐步增长。

再次强调：对 5K/10K、半马/全马，Fundamental 期后半段会从 有氧耐力区 走向 有氧动力区。

## 10K Specific 课表

下面课表按 Canova 的 Specific 原则写：核心速度是 `10K pace` 的 95-105%。

Canova 原帖给的 精英跑者 示例包括：

- `15 x 600`
- `10 x 1000`
- `4 x 2000 + 1 x 1000`
- `3000 + 2 x 2000 + 4 x 1000`

| 类型 | 课表 | 目的 |
|---|---|---|
| 入门 specific | `12-15 x 600m @ 10K pace，rec 60-90s jog` | 建立 10K pace 触感 |
| 标准 specific | `8-10 x 1000m @ 10K pace，rec 90s-2min jog` | 累积接近比赛距离的专项量 |
| 延展 specific | `4-5 x 2000m @ 10K pace，rec 3-4min jog` | 提高单段耐受 |
| 组合课 | `3000m @ 10K pace + 2 x 2000m @ 10K pace + 4 x 1000m @ 10K pace/slightly faster` | 模拟比赛后半段压力 |
| 连续课 | `6-8km continuous @ 95-100% 10K pace` | 提高节奏稳定性 |
| 峰值课 | `3 x 3000m @ 10K pace，rec 3min jog` | 接近比赛特异性，负荷较高 |

10K 的 Specific 总量可以接近比赛距离，甚至略高，但不能每周堆。关键课之间要有足够恢复。

## 半马 Specific 课表

Canova 原帖对 HM 给的例子包括：

- `7 x 2000`
- `5 x 3000`
- `3 x 5000`
- `15km @ 102% HMP`
- `25km @ 97% HMP`

| 类型 | 课表 | 目的 |
|---|---|---|
| 入门 specific | `6-7 x 2000m @ HMP，rec 400m float/jog` | 建立 HMP 重复能力 |
| 标准 specific | `5 x 3000m @ HMP，rec 800-1000m float` | 提高 HMP 下的耐受 |
| 延展 specific | `3 x 5000m @ HMP，rec 1000m float` | 接近半马决定性训练 |
| 持续快跑课 | `15km @ 102% HMP` | 比 HMP 略快，控制总距离 |
| 长距离专项 | `22-25km @ 95-97% HMP` | 比 HMP 稍慢，延展持续时间 |
| 混合课 | `4 x 4km @ HMP + 1km float` | 更接近半马节奏波动 |

半马 specific 的重点是：既要能在 HMP 附近跑长段，也要能在稍慢于 HMP 的速度下跑更长。

对业余跑者，`3 x 5km @ HMP` 已经是非常硬的课，不应频繁使用。

## 全马 Specific 课表

Canova 原帖给马拉松的例子非常典型：

- `6 x 4000`
- `5 x 5000`
- `4 x 6000`
- `4 x 7000`
- `25km @ 102% MP`
- `30km @ MP`
- `35km @ 97% MP`
- `40km @ 92% MP`

这些课的恢复不是完全站休，而是 1000m 较快恢复，或者直接用连续跑完成。

| 类型 | 课表 | 目的 |
|---|---|---|
| 入门 specific | `6 x 4km @ MP-102% MP，rec 1km float` | 建立长段 MP 经济性 |
| 标准 specific | `5 x 5km @ MP，rec 1km float` | 高专项总量，接近马拉松需求 |
| 延展 specific | `4 x 6km @ MP，rec 1km float` | 提高单段持续能力 |
| 大段 specific | `4 x 7km @ 97-100% MP，rec 1km float` | 高负荷，接近峰值课 |
| 连续 MP | `30km @ MP` | 直接验证目标马配能力 |
| 稍快连续 | `25km @ 102% MP` | 比目标马配略快，距离较短 |
| 稍慢长距离 | `35km @ 97% MP` | 延展马拉松耐力 |
| 超长支撑 | `38-40km @ 90-92% MP` | 模拟比赛的耐力支撑 |

对全马来说，Specific 课通常短于比赛距离。

难点在于：长距离、接近 MP、恢复不完全，并且前后训练还能承受。

## 三种比赛的专项差异

10K、半马、全马都遵循 95-105% race pace，但训练形态不同。

| 项目 | Specific 总量 | 单段长度 | 恢复方式 | 最关键能力 |
|---|---:|---:|---|---|
| 10K | 接近或略高于 10km | 600m-3000m | 短 jog / 短时间恢复 | 在高乳酸压力下维持 race pace |
| 半马 | 通常 14-25km | 2km-5km / 15-25km continuous | **float** 更重要 | 长时间维持 HMP 的稳定性 |
| 全马 | 通常 25-40km | 4km-7km / 25-40km continuous | 1km **float** 或连续跑 | MP 经济性、糖原节省、后程稳定 |

## 一句话总结

Canova 训练理念可以压缩成一句话：

> 先建立足够的有氧和速度支撑，再围绕目标比赛速度，用 extension 的方式把专项能力延展到比赛距离，同时用充分的 regeneration 保证关键课质量。

千万不要一根筋地认为长跑训练是跑量至上或者强度至上。

Aerobic ! Support ! **Race pace** ! **Extension** ! Regeneration!
